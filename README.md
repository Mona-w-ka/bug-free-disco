# Visual2D

Программа визуализации на С++ простых 2D объектов на плоскости с помощью Qt. 

# Для запуска требуется 

QCustomPlot 2.1.1

Qt 6.x

C++ 11 и выше



# Порядок запуска #

1. Клонировать репозиторий git clone https://github.com/Mona-w-ka/bug-free-disco
2. Открыть проект в QtCreator
3. Запустить сборку проекта
4. Запустить проект
